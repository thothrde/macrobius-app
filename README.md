# Macrobius App

Eine öffentliche Version der interaktiven Web-Anwendung über Macrobius.

## Features

- Interaktive Weltkarte mit historischen Ansichten
- Kosmologische Visualisierungen
- Textanalyse und -verarbeitung
- Mehrsprachige Unterstützung (DE/EN/LA)
- Responsives Design

## Technologie-Stack

- Next.js
- TypeScript
- Tailwind CSS
- Lucide Icons
- shadcn/ui Components

## Installation

1. Repository klonen:
\`\`\`bash
git clone https://github.com/thothrde/macrobius-app.git
\`\`\`

2. In das Projektverzeichnis wechseln:
\`\`\`bash
cd macrobius-app
\`\`\`

3. Abhängigkeiten installieren:
\`\`\`bash
npm install
\`\`\`

4. Entwicklungsserver starten:
\`\`\`bash
npm run dev
\`\`\`

5. Browser öffnen und http://localhost:3000 aufrufen

## Verzeichnisstruktur

- \`/public/assets/\` - Bilder und Videos
- \`/src/components/\` - React-Komponenten
- \`/src/lib/\` - Utilities
- \`/src/pages/\` - Next.js Seiten
