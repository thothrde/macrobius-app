# Macrobius App

Eine interaktive Web-Anwendung über Macrobius und seine Werke.

## Features

- Interaktive Weltkarte mit historischen und modernen Ansichten
- Quiz-System zur Überprüfung des Wissens
- Mehrsprachige Unterstützung (DE/EN)
- 3D-Visualisierungen

## Technologie

- Frontend: Next.js
- Styling: Tailwind CSS
- UI-Komponenten: shadcn/ui
- Karten: SVG + Interaktionen

## Installation

1. Repository klonen
2. `npm install` ausführen
3. `npm run dev` für Entwicklungsserver
4. Im Browser unter `http://localhost:3000` öffnen
